# DejaVu Sans font

Este directorio almacena recursos de tipografía utilizados por `visualize_keypoints_viewer.py`.

- **Fuente**: [DejaVu Sans](https://dejavu-fonts.github.io/)
- **Versión**: 2.37
- **Licencia**: Licencia de fuentes Bitstream Vera / Dominio público. Consulta el archivo `LICENSE` incluido en el repositorio del proyecto para más detalles.

La fuente se distribuye aquí para garantizar que caracteres con acentos y otros signos diacríticos se muestren correctamente en el visor, incluso en sistemas que no tienen instaladas fuentes compatibles de manera predeterminada.
